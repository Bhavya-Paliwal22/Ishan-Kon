# 🏡 ISHAN KON – Interior Design Web Platform

**Ishan Kon** is a modern interior design showcase platform created to highlight elegant home concepts, premium room layouts, and personalized living spaces.  
This project focuses on minimal aesthetics, structured layouts, and a smooth user experience that reflects high-end interior branding.

---

## ✨ Features

- Clean and modern UI/UX  
- Image-based layout showcase  
- Responsive and mobile-friendly design  
- Smooth transitions & animations  
- Organized content sections for better readability  

---

## 🎯 Project Objectives

- Present interior concepts in a visually appealing format  
- Provide a premium browsing experience  
- Demonstrate UI/UX skills using modern web technologies  
- Build a clean portfolio-style interior website  

---

## 🛠️ Tech Stack

- **HTML**  
- **CSS**  
- **JavaScript**
